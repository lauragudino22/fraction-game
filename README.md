Fraction Tic-Tac-Toe
An educational math game that combines the classic game of tic-tac-toe with fraction practice! Players must solve fraction problems correctly to place their X or O on the board.

## Game Features

- **Three Difficulty Levels:**
  - **Easy** - Adding and subtracting fractions with the same denominator
  - **Medium** - Adding and subtracting fractions with different denominators
  - **Hard** - Adding and subtracting mixed numbers with fractions

- **Two-Player Mode** - Play against a friend locally
- **Score Tracking** - Keep track of wins across multiple rounds
- **Flexible Answers** - On hard mode, students can answer as whole numbers, mixed numbers, or improper fractions
- **Fully Responsive** - Works on phones, tablets, and computers
- **No Installation Required** - Just open the HTML file in any web browser
